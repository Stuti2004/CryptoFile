# CryptoFile
A web-based file encryption and decryption tool built using JavaScript, CryptoJS, and Tailwind CSS.

# Features

## AES Encryption/Decryption:
Strong client-side encryption using CryptoJS AES.

## No backend required:
Fully browser-based, works offline.

## User-friendly UI:
Modern and responsive design with Tailwind CSS.

## File download:
Encrypted and decrypted files are downloaded automatically.

## Visual feedback:
Success/error messages guide the user.

## Cross-platform:
Works on any modern browser and device.

# How to Use

## Select a file to encrypt or decrypt.

## Enter a secure password.

## Click Encrypt or Decrypt.

## The resulting file will automatically download.
